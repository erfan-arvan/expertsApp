Phase 2: Ranking Code Snippets by Comprehensibility

In this phase, you will review all 8 code snippets and rank them based on how easy or difficult they are to understand.

  1. Rank the snippets: Drag and drop each snippet card into one of the 8 ranking positions below.
  2. Understand the scale:
    • Position 1 (leftmost): Most comprehensible snippet.
    • Position 8 (rightmost): Least comprehensible snippet.
  3. Equal ranking: You may place multiple snippets in the same position if they are equally comprehensible.
  4. Preview feature: Use the checkbox on each card to view the full code below.

Important: Please fill the ranking positions from left to right. If you leave any positions empty (e.g., due to grouping multiple snippets in one position), leave the rightmost positions empty.

Prohibited: Please do not use generative AI tools (e.g., ChatGPT, GitHub Copilot) during this phase. Your expert judgment is essential to the validity of our study
